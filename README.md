# School Finance Dashboard

Dashboard created to organize school income, expenses and financial records in a simple, visual and functional way.

## Overview

This project was built based on a real need observed in school management: many schools do not have a clear financial control over income, expenses and funding sources.

The goal was to create an intuitive and accessible interface that could be used by school managers, even with limited familiarity with digital systems.

## Features

- Add income and expense records
- Register source of funds
- Display current balance
- Display total income
- Display total expenses
- Transaction listing
- Delete transaction
- Clear all records
- Local data persistence with localStorage
- Automatic currency formatting
- Visual feedback after adding records
- Sorting by date

## Technologies

- HTML5
- CSS3
- JavaScript
- LocalStorage

## Technical highlights

- DOM manipulation
- Form events
- Dynamic rendering
- Responsive layout

## Context

This project was designed as a practical solution for Brazilian schools, considering a real scenario where institutions often struggle to organize resources from events, tuition, donations, public funds and other expenses.

## How to run

1. Download the project files
2. Open `index.html` in your browser
